## 一句话定位

**云图是什么**：为中文 AI 用户提供"发现 + 试用 + 交流"的平台。
**云图不是什么**：不是 AI 工具目录站（Toolify），不是纯内容平台（少数派），不是通用 API 中转站。
**终局想象**：AI 时代的中文 App Store + Steam。
