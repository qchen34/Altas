品牌内核

ESSENCE · WHO WE ARE

云图（Atlas）是一个为中文 AI 用户提供「发现、试用、交流」的平台，服务三类人——做产品的 indie 开发者、解决具体问题的普通用户、追踪趋势的高密度信息消费者。

它不是工具目录，是一张正在被绘制中的世界地图。

我们的生命从不只属于自己——每一次善意的举动，都会化作涟漪，穿越世界，穿越时间。 "Our lives are not our own. By each act of kindness, we ripple across the world and through time."

05

动效语言

MOTION · A FILM IN MOTION

云图的动效遵循电影感——慢、有重量、有迟滞、避免回弹。所有过渡都基于「缓出」（ease-out）的非对称曲线；所有进入都伴随轻微 Y 轴下沉到归位（28px），让画面像胶片掠过印刻而成。

REVEAL · 主入场 1200ms · cubic-bezier(.2,.7,.2,1)
translateY(28px) + opacity 0 → 归位 + opacity 1。带 staggered delay 60–500ms。所有正文、标题、卡片均使用此动效。
FADE-ONLY · 纯渐显 1600ms · ease-out
仅透明度变化，不带位移。用于 Hero 主词、引用、字幕等需要"轻盈出现"的场景。
BLUR-IN · 失焦聚焦 1400ms · ease-out
blur(12px) + opacity 0 + Y(20px) → 归位。用于章节首屏，营造电影从模糊聚焦的镜头感。
PULSE · 心跳 2000ms · ease-in-out infinite
微小的明灭（0.4 → 1 → 0.4）+ scale (0.85 → 1)。仅用于状态指示点（HUD 的 REC 圆点）。
CONVERGE · 汇聚 ~3200ms · ease-in-out
点云从随机散点 → 目标位置（圆环）→ 邻近点连线。专属动效，仅出现在结尾的 Atlas 星图。

07

语调原则

VOICE · HOW WE TALK

云图的文字是克制的——它不喊叫、不感叹、不做承诺。它陈述事实、引用台词、用电影感的中英对照展开节奏。每一段文字都应能被放入字幕条而不显违和。

DO · 这样写
  • 说人话，少用商业黑话（"被发现的地方" 而不是 "增长触点"）
  • 用具体瞬间替代抽象概念（"周二晚上 23:47"）
  • 中英并置，营造电影台词感（中文台词 + 英文 caption）
  • 留白，让句子单独出现（一句话一行，独占一段）
  • 尊重读者智商，不解释显而易见的事
DON'T · 不要这样
  • 感叹号、表情符号（除非必须，几乎不用）
  • "赋能"、"重塑"、"颠覆"等空词
  • 过长复合句、堆砌形容词
  • 渐变色、霓虹紫粉、广告体配色
  • 圆角、不必要的阴影、霓虹毛玻璃（App 层内的毛玻璃见第 09 节规范）
08

使用守则

USAGE · GUARDRAILS

本品牌系统是开放的，但有几条不能动的红线——它们决定了云图被一眼认出的能力。

01 · ALWAYS
白天模式为主
默认使用白底 + 黑色标识，类似 App Store 风格；深色底仅在夜间模式或深色背景环境下使用。
02 · ALWAYS
永远 0 圆角
所有容器、卡片、按钮一律方角。圆角属于消费产品，不属于云图。
03 · ALWAYS
永远 1px
所有线条、边框、分隔均为 1px hairline。粗线属于广告，不属于云图。
04 · NEVER
不引入新色
只用本手册定义的 6 种色 + 透明度变体。新色需要重新版次审定。
05 · NEVER
不堆砌 emoji
如必须传达状态，使用方块符号（▮ · ▢ · ◌ · ✓ · ⧉）替代 emoji。
06 · NEVER
不旋转/倾斜文字
所有文字保持水平。倾斜版式属于潮牌，不属于地图集。

毛玻璃规范

FROSTED GLASS · APP UI SPEC

毛玻璃是Z 轴工具，不是装饰。每个层级的 blur 值与不透明度都经过配对调校——blur 越大，背景遮蔽越重，以保证文字可读性。saturate 增强让星蓝底色穿透玻璃层，维持品牌色感知。

亮色模式变体 · LIGHT MODE

亮色背景下，毛玻璃改用白色半透明 + 蓝色边线——维持同等层次感，同时避免在明亮场景中显得沉重。brightness(1.04) 防止玻璃层在白色背景下"消失"。

动效过渡规范 · GLASS MOTION

毛玻璃层的进出场动效与主系统动效语言保持一致——缓出曲线、有重量感——并在此基础上叠加 blur 从模糊到清晰的「聚焦」效果，强化层次感的出现感知。

M1 · GLASS-IN
浮层 / 卡片 入场
600ms · cubic-bezier(.2, .7, .2, 1)
opacity 0→1 + translateY(24px)→0 + blur(12px)→0。三属性同步，blur 聚焦感营造"从背景升起"的质感。
animation: glassIn 600ms cubic-bezier(.2,.7,.2,1) both; @keyframes glassIn { from { opacity:0; transform:translateY(24px); filter:blur(12px); } to { opacity:1; transform:translateY(0); filter:blur(0); } }
M2 · GLASS-OUT
浮层 / 卡片 退场
380ms · cubic-bezier(.4, 0, .6, 1)
opacity 1→0 + translateY(0)→-16px + blur(0)→6px。退场比入场快 40%，避免用户等待。
animation: glassOut 380ms cubic-bezier(.4,0,.6,1) both; @keyframes glassOut { from { opacity:1; transform:translateY(0); filter:blur(0); } to { opacity:0; transform:translateY(-16px); filter:blur(6px); } }
M3 · SHEET-UP
底部抽屉 升起
520ms · cubic-bezier(.2, .7, .2, 1)
translateY(100%)→0 + opacity 0→1。纯位移，不叠加 blur——Sheet 层级最高，不需要聚焦感，只需重量感。
animation: sheetSlideUp 520ms cubic-bezier(.2,.7,.2,1) both; @keyframes sheetSlideUp { from { transform:translateY(100%); opacity:0; } to { transform:translateY(0); opacity:1; } }
M4 · SHEET-DOWN
底部抽屉 收起
360ms · cubic-bezier(.4, 0, .6, 1)
translateY(0)→100% + opacity 1→0。收起动效快且干脆，参照 iOS Sheet dismiss 节奏。
animation: sheetSlideDown 360ms cubic-bezier(.4,0,.6,1) both; @keyframes sheetSlideDown { from { transform:translateY(0); opacity:1; } to { transform:translateY(100%); opacity:0; } }
M5 · NAV-BLUR-IN
导航栏 页面滚动激活
transition: 280ms · ease-out
页面从顶部滚动超过 8px 时，导航栏毛玻璃从透明 → 完整效果。使用 transition 而非 animation，与滚动实时联动。
/* 初始：透明导航栏 */ .navbar { background: transparent; backdrop-filter: none; transition: background 280ms ease-out, backdrop-filter 280ms ease-out; } /* 滚动后激活 */ .navbar.scrolled { background: rgba(5,6,18,0.62); backdrop-filter: blur(28px) saturate(160%); }
M6 · STAGGER
多卡片交错入场
delay: n × 60ms
列表中多个玻璃卡片依次入场时，每张卡片的 animation-delay 以 60ms 递增，形成"逐一浮现"的视觉节奏。
/* 每张卡片叠加 delay */ .glass-card:nth-child(1) { animation-delay: 0ms; } .glass-card:nth-child(2) { animation-delay: 60ms; } .glass-card:nth-child(3) { animation-delay: 120ms; } .glass-card:nth-child(n) { animation-delay: calc((n - 1) * 60ms); }
动效原则 · Motion Principles
01 · 入慢出快
入场 600ms，退场 360–380ms。符合人类注意力分配——入场需感受重量，退场需干净利落。
02 · blur 只用于入场
blur 聚焦感仅在进入时使用（glassIn），退场使用轻微 blur(6px) 即可，避免过度设计感。
03 · 禁止回弹
所有毛玻璃动效禁止使用 spring / bounce 曲线。弹性属于活泼产品，云图追求电影感的沉稳。
