# 云图 · 用户画像

> 云图精确服务三类人。每一类都有清晰的"为什么来"、"为什么留"、"为什么转介绍"。
> 这份文件用于：做产品决策时对齐用户、写评测时确认受众、设计功能时找优先级。

---

## 用户 A · Indie 开发者 / Vibe Coder

**角色定义**：用 AI 编程工具（Cursor / Claude Code / Lovable / Bolt）独立做产品的个人开发者。通常单兵作战，产品是 web app 形态。

**人口特征**：
- 年龄：20-35 岁为主
- 技术背景：会写代码，但不一定是专业程序员
- 产品阶段：demo / MVP / 早期产品
- 常出没地点：即刻、V2EX、推特、GitHub

**核心痛点（按重要性）**：

1. **没有曝光渠道** · 做出来没人看，朋友圈没人点赞，Product Hunt 英文门槛高
2. **拿不到真实反馈** · 不知道用户卡在哪一步，单兵作战没有用研能力
3. **无法快速迭代验证** · 想知道 v1 和 v2 哪个更好，但没有数据对比
4. **个人品牌建立难** · 作品散落各处，没有一个持续累积的主页
5. **商业化路径不清晰** · 想收钱但不知从哪开始，支付工具复杂

**在云图的使用场景**：
- 发布新产品，获得首批真实用户和反馈
- 查看数据看板（谁试用了、停留多久、在哪步放弃）
- 发布 v2/v3 迭代版本，对比数据变化
- 建立个人作品集主页

**他们为什么会付费**：
- 首发推广服务（$299-999）= 花小钱买"真实早期用户 + 反馈"，比任何广告都精准

**获取渠道**：
- GitHub: `1c7/chinese-independent-developer`
- Solo 社区 / Mergeek
- 即刻（搜"vibe coding"、"独立开发"、"indie"）
- V2EX（程序员版块）
- Twitter 中文 AI 圈

## 用户 B · 普通用户 / 解决具体问题的人

**角色定义**：在学习或工作中遇到了一个具体问题，希望用 AI 工具解决，但不知道选哪个。

**典型画像**：
- **场景 A**：品牌设计大学生，要在 24 小时内做一组品牌样机看板（logo 在 T 恤 / 咖啡杯 / 名片上的 mockup），不知道用哪个 AI 工具最合适
- **场景 B**：公司运营，有 50 封客户邮件要提炼需求，想用 AI 帮忙，但不知道哪个工具最适合这个场景
- **场景 C**：考研党，想用 AI 帮助整理笔记，但打开 Toolify 看到 800 个工具，无从下手

**核心痛点**：
1. 有非常具体的场景需求，但工具选择困难症
2. 不想为"试错"付费（要先看到效果再决定）
3. 使用门槛高（要翻墙 / 要英文 / 要美元付费）
4. 功能分类对他们没用（他们思考的是"我要解决什么问题"，不是"我要用什么类型的工具"）

**在云图的使用场景**：
- 搜索"品牌样机生成"，直接看到精准推荐 + 试用按钮
- 在站内试用工具（不跳出、不注册海外账号）
- 看"什么人该用、什么人别用"的场景说明
- 看同类用户的真实反馈

**他们为什么会留**：
- 找到了以前找不到的精准工具（首次价值）
- 每次有新需求都能找到，形成习惯（留存价值）

**获取渠道**：
- 小红书（搜"AI 工具推荐"、"设计工具"等）
- 知乎（搜具体场景问题）
- 微信群自然传播
- SEO（按场景关键词）

## 用户 C · 高密度信息消费者

**角色定义**：主动追踪 AI 行业动态的重度信息消费者。包括早期投资人、科技媒体人、连续创业者、AI 从业者、技术 KOL。

**核心特征**：
- 每天花大量时间消费科技信息
- 订阅多个英文 newsletter（The Information / Stratechery / tldr.tech 等）
- 有转发分享优质信息的习惯
- 社交资本高：他们一条转发，影响力是普通用户的 10-100 倍

**核心痛点**：
1. 缺一个有判断力的中文信息源（英文信息源信息密度高，但语言门槛存在）
2. 不想被 SEO 垃圾内容浪费时间（"又是一篇凑字数的 AI 工具推荐文"）
3. 想找到投资机会 / 合作机会 / 学习机会
4. 想展示自己的信息品味（转发云图 = 我有 taste）

**在云图的使用场景**：
- 订阅每周精选邮件（每周固定来看）
- 发现一个有潜力的 indie 产品，转发到自己的圈子
- 把云图推荐给需要找工具的朋友

**他们的核心价值（对云图）**：
- 放大器：他们的转发带来其他两类用户
- 背书：他们的认可让云图建立"有 taste"的品牌

**他们为什么会付费**：
- Pro 订阅（¥299/年）= 提前看到内容 + 独家深度分析，物超所值

**获取渠道**：
- 推特（找 AI 圈中文 KOL）
- 即刻（科技圈活跃用户）
- 微信朋友圈（口碑传播）
- 邮件订阅（主动找来的）

## 三类用户的关系

```
用户 A（开发者）
      ↓ 发布产品
     云图
      ↓ 精选 + 推荐
用户 B（普通用户） ← 试用 → 付费给开发者
      ↑
用户 C（信息消费者）→ 转发 → 更多用户 B
```

**飞轮逻辑**：
- 开发者发布 → 云图精选推荐 → 用户 B 发现并试用 → 用户 C 看到有价值的内容并转发 → 更多用户 B 涌入 → 开发者获得更多用户和数据 → 开发者持续发布更好的产品
